Programación Orientada a Objetos

La programación orientada a objetos (Object Oriented Programming, OOP) es un paradigma que organiza el software mediante la creación y manipulación de "objetos". Estos objetos son instancias de clases, que definen estructuras de datos y comportamientos.

Encapsulamiento

Agrupa datos y métodos en una clase y controla el acceso a ellos.

Herencia

Permite que una clase herede características de otra.

Polimorfismo

Permite que métodos con el mismo nombre se comporten de manera diferente según el objeto.

Abstracción

Oculta detalles internos y muestra solo lo esencial del objeto.
